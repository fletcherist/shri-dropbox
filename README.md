# shri-dropbox

Бесплатное CORS-хранилище для твоих видео, аудио и субтитров. 



## API

Напиши адрес, который тебя интересует после 

# http://146.185.134.219/${URL}



### Contribute

Система развёртывается

git clone

npm install

pm2 start app.js



Подними свой сервер, если на этом кончится место.
